# Eclavine – Women's Fashion E-Commerce Website

Eclavine is a modern, responsive women's fashion e-commerce website built using **HTML, CSS, and JavaScript** in a single `index.html` file. The project features a clean and elegant design, a responsive layout, and a fully functional shopping cart.

## ✨ Features

* Responsive design for desktop, tablet, and mobile
* Elegant women's fashion UI
* Hero banner
* Product showcase
* Add to Cart functionality
* Shopping cart sidebar
* Increase and decrease product quantity
* Live cart item count
* Automatic total price calculation
* Modern and minimal design
* Single-file implementation (HTML, CSS, and JavaScript)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/eclavine.git
```

2. Open the project folder.

3. Double-click `index.html` or open it in your preferred web browser.

No installation or additional dependencies are required.
